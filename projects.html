HIE Treestocker App

Unit: Professional Experience

Led a semester-long project to transform the Hawkesbury Institute for the Environment's Tree Stock Standard Website. Utilizing the AS2303 standard, the HIE had developed an Online Tree Assessment Tool for visualizing tree compliance. Our team's mission was to replicate this Rshiny Website as an offline app using Dart and Flutter, with platform-specific development done in Android Studio and Xcode. We maintained version control via GitLab.

The app mirrors the functionalities of the original website, including:

  • Home Page: A simple welcoming page introducing the tool and the website that facilitated graph creation.
  • Data Input: Users can input data into three fields (Volume, Calliper, and Height).
  • Graph Output: Based on these inputs, the app generates a dot on a graph illustrating compliance with the AS2303 stock standard.
  • Database: The app displays a map with markers representing nurseries contributing to the AS2303 Tree Stock Standard database.

SDES and RSA Algorithms

Unit: Information Security

For my Information Security unit, I was tasked with Recreating RSA and SDES cryptographic algorithms in C++. These programs would allow users to input their data, which was then subjected to cryptographic processes based on their respective algorithms.
RSA Program: I made sure to use modular exponentiation to keep the power equations running efficiently and to save on memory usage. I also had to use modular arithmetic so that the mathematical operations involved remained within manageable bounds, ensuring the proper functioning of the encryption and decryption processes.

SDES and RSA Algorithms

Unit: Information Security

S-DES Program: I followed a well-defined sequence of cryptographic functions using specified arrays as guides. These arrays, like P10, P8, E/P, and S-boxes (S0 and S1), served as essential blueprints for key generation, initial permutation, expansion permutation, and substitution operations. All of these functions are based on the original SDES permutations and substitution operations.

Hex Strategy Game in C++

Unit: Data Structures and Algorithms

My Data Structures and Algorithms Assignment 1 tasked me with recreating a program of the board game “Hex”. More information can be found here. We were restricted to C++ and had to stick with text-based console output for our assignment. While we had the freedom to choose any algorithm, our grades were influenced by how efficient it was. I decided to go with a best-first search to help the AI opponent find the longest path. To determine if a player won, I used a stack to compare the cells and coordinates, checking if they matched with the ones around them. If a connection linked one side's cell to the other, that player won.
